# Contributing to RepoGardener

Thanks for your interest in improving RepoGardener! 🌱

## How to Contribute

1. Fork the repo
2. Create a feature branch (`git checkout -b feat/amazing-feature`)
3. Make your changes
4. Run `ruff check .` and `python -m pip install -e . && repo-gardener scan .`
5. Commit your changes (`git commit -m 'feat: add amazing feature'`)
6. Push to the branch (`git push origin feat/amazing-feature`)
7. Open a Pull Request

## Development Setup

```bash
git clone https://github.com/troythedecoyx/repo-gardener.git
cd repo-gardener
python -m pip install -e .
ruff check .
```

## Code Style

- Use Ruff for linting and formatting
- Keep functions small and focused
- Add helpful suggestions in the CLI when improving scores

## Reporting Issues

Please use the [GitHub Issues](https://github.com/troythedecoyx/repo-gardener/issues) page.

We appreciate all contributions, big or small!
